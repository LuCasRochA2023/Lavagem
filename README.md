# FTW Soluções automotias
Site de lavagem e estacionamento 
O site serve para Gerenciar um sistema de agendamento de lavagem de carros, agendando horario e carro a ser lavado a partir dos dados do cliente cadastrados no sistema.
O site utiliza os Framework para o Front-End: React.js, e BootsTrap utilizando as seguintes dependências:  
        "@emotion/react": "^11.13.3",
        "@emotion/styled": "^11.13.0",
        "@hookform/resolvers": "^3.9.0",
        "@mui/material": "^6.1.6",
        "@react-icons/all-files": "^4.1.0",
        "classnames": "^2.5.1",
        "react-hook-form": "^7.53.1",
        "react-icons": "^5.3.0",
        "react-router-dom": "^6.27.0",
        "sass": "^1.80.3",
        "zod": "^3.23.8".
No Back-end será utilizado: PHP, MySQL e as seguintes dependência:
        "phpunit/phpunit": "^11.3"
